# Репозиторий бекенда для сайта по расчету доступности системы
Стек: 
Go, GIN, Gorm

---

## Описание структуры репозитория

- `main` — главная ветка  
- `backend-template` — MVP версия
- `backend-database` — инизцициализация БД
- `api-backend` — добавление основных API методов
- `api-auth-and-docs` — добавление системы авторизации через JWT и документации Swagger
